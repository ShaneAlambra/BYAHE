PAANO MAGLAGAY NG LARAWAN
=========================

Bawat folder dito ay para sa isang larawan sa website.
I-drag/copy mo lang ang larawan mo sa TAMANG folder — automatic na ito
lalabas sa site. KAHIT ANONG pangalan ng file (1.jpg, IMG_2025.png, etc.).
Kung marami kang ilalagay sa isang folder, ang UNA (alphabetical) ang lalabas.

Tinatanggap: .jpg .jpeg .png .webp .avif .gif

ANG MGA FOLDER AT SAAN SILA LUMALABAS:

  hero-photo/     -> malaking larawan sa taas (hero)
  kami-photo/     -> "Shane & Hazel" sa About Us
  motor-photo/    -> "our trusty ADV" sa About Us
  favorite-photo/ -> "our favorite ride" section

  GALLERY (one per destination):
  dest-bolinao/    -> Bolinao, Pangasinan
  dest-dasol/      -> Dasol, Pangasinan
  dest-rosario/    -> Rosario, La Union
  dest-sanjuan/    -> San Juan, La Union
  dest-tagaytay/   -> Tagaytay, Cavite
  dest-marilaque/  -> Marilaque, Rizal
  dest-infanta/    -> Infanta, Quezon
  dest-drt/        -> DRT, Bulacan
  dest-angat/      -> Angat, Bulacan
  dest-sanrafael/  -> San Rafael, Bulacan
  dest-sjdm/       -> SJDM, Bulacan

  STORIES (Stories from the Road):
  kwento-1/  -> "Our very first getaway" (Bulacan)
  kwento-2/  -> "To the end of the twisties" (Marilaque-Infanta)
  kwento-3/  -> "3AM departures, northbound" (Pangasinan & La Union)

TANDAAN:
- Buksan ang site sa pamamagitan ng http://localhost/BYAHE/ (hindi
  double-click sa file) para gumana ang auto-load.
- Kung mag-drag-drop ka pa rin ng larawan diretso sa slot sa website,
  iyon ang mananaig kaysa sa larawan sa folder.
